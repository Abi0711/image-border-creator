# Image Border Creator
## What is in this project?
Upload a PNG image for the easy creation of a border around the uploaded image. After you've created the border and it suits your preference, you can download it at the click of a button. 

[Available here!](https://abi0711.github.io/sticker-border-studio/)

![Main page](/images/main_page.png)

Supports uploading and editing multiple files.

![Multiple Files Example](/images/multiple_files.png)

Inspiration behind creating this is due to Cricut adding a paywall to its create border feature.


## 🚀 Quick Start

### 1. Access the website through your browser

[Available here!](https://abi0711.github.io/sticker-border-studio/)

### 2. Upload files

Drag and drop files or click anywhere in the dotted border box to access the file explorer.

### 3. Edit Border

Choose the border thickness using the slider or input box. Choose the desired border colour. by clicking inside the coloured box next to <strong>Border Colour:</strong>

### 4. Download Image(s)

Choose to either download the current image you are working on

OR

Apply the border that you've designed for the current image, to all the images uploaded and download them all.
